# 투자 전략 룰

## 기본 원칙
- 워런 버핏 투자 철학 기반, S&P 500 코어 장기 보유
- 단기 뉴스에 흔들리지 않고 장기 관점 유지
- 룰 베이스 매수는 **449180(환헤지)만** 실행 (환율 고려 불필요)
- 정기매수는 ISA 활성 종목(환율 기준 360750/449180 전환)으로 진행

## 환율 정책

### 구간별 환전 액션
| 환율 구간 | 구간명 | 월 환전액 |
|-----------|--------|-----------|
| ~1,380 미만 | 전액 환전 | 쌓인 원화 전액 |
| 1,380~1,420 | 정상 (전액) | 80만원 |
| 1,420~1,450 | 정상 (절반) | 40만원 |
| 1,450 초과 | 보류 | 0원 (원화 적립) |

### ISA 활성 종목 전환
- 환율 >= 1,380원 → **449180** (KODEX 미국S&P500(H), 환헤지)
- 환율 < 1,380원 → **360750** (TIGER 미국S&P500, 환노출)
- 정기매수에만 적용, 룰 베이스 매수와는 무관

## ISA 매수 트리거 (449180 전용)

### 전월 말일 대비 (기존)
| 조건 | 액션 |
|------|------|
| 전월 대비 -5% | 현금 버퍼에서 100만원 추가 매수 |
| 전월 대비 -10% | 현금 버퍼에서 100만원 추가 매수 |

### 2달 전 대비 (slowly melting 방지, 구현 예정)
| 조건 | 액션 |
|------|------|
| 2달 전 대비 -5% | +50만원 추가 매수 |
| 2달 전 대비 -10% | +100만원 추가 매수 |

- 전월 트리거와 독립적으로 동작 (중복 발동 가능)
- 449180은 환헤지 상품이므로 환율 변동 무관

## ISA 매도 트리거 (133690 전용)

TIGER 미국나스닥100 (133690.KS) 전월 말일 대비:
| 조건 | 액션 |
|------|------|
| +5% 이상 상승 | 50% 매도 (보유량의 절반) |
| +10% 이상 상승 | 전량 매도 |

## QCOM 조건부 매수

두 조건 모두 충족 시 매수 신호:
- PER <= 25
- 52주 고점 대비 15% 이상 하락

## 현금 관리
- ISA 계좌 현금과 토스증권(원화+달러)을 분리 관리
- config.yaml의 cash 값은 **매달 1일** 실제 잔액으로 수동 업데이트
- 달러 현금은 실시간 환율로 원화 환산하여 총자산에 합산
